API Quickstart

Call the API with curl. Replace the host with yours if self-hosting; YOUR_API_KEY is filled in automatically when you are logged in. 

curl -X POST https://getsnapapi.uk/screenshot \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com"}'
curl -X POST https://getsnapapi.uk/pdf \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com","width":1280,"height":720}'

Optional width and height must be sent together: they set the browser viewport and the PDF page size. Omit both to use the document’s scroll dimensions.

Request body

Field Type Required Notes
url string required Page to load.
fullPage boolean — Screenshot only: capture full scrollable page (default true). Set false for viewport-only.
width integer — Viewport width (320–4096). Only applied if height is also set.
height integer — Viewport height (240–4096). Together with width, defines PDF page dimensions.

Success response (HTTP 200) 

{
  "status": "success",
  "message": "Screenshot saved",
  "path": "https://your-bucket.r2.dev/screenshot-1712345678901.png"
}

For PDF, message is "PDF saved"; path is a public HTTPS URL to the file.
