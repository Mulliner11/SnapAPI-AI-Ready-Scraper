API Quickstart

Use curl to call the screenshot endpoint:

curl -X POST https://snapapi.up.railway.app/screenshot \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com"}'
