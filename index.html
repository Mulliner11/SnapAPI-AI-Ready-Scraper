API Quickstart

POST /api/scrape with x-api-key. Replace the host when self-hosting; YOUR_API_KEY is injected when you are logged in. 

curl -X POST https://getsnapapi.uk/api/scrape \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com"}'

Request body

Field Type Required Notes
url string required HTTP(S) page to fetch and extract.

Success response (HTTP 200) 

{
  "title": "Example Domain",
  "markdown": "# Example Domain\\n\\n…",
  "text_content": "Example Domain This domain is …"
}
