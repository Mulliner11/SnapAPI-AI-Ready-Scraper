API Quickstart

使用 curl 调用接口；将示例中的域名与 YOUR_API_KEY 换成你的环境值。 

curl -X POST https://getsnapapi.uk/screenshot \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com"}'
curl -X POST https://getsnapapi.uk/pdf \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"url":"https://example.com","width":1280,"height":720}'

可选 width 与 height 需成对传入，用于设置浏览器视口并作为 PDF 页面尺寸；省略时按页面滚动尺寸自动计算。

请求体参数

参数 类型 必填 说明
url string required 要打开的页面地址。
fullPage boolean — 仅 Screenshot：是否截取整页，默认 true；传 false 则仅当前视口。
width integer — 视口宽度（320–4096）；须与 height 同时传入方生效。
height integer — 视口高度（240–4096）；PDF 模式下与 width 共同决定导出页面尺寸。

成功响应 (HTTP 200)

{
  "status": "success",
  "message": "Screenshot saved",
  "path": "https://your-bucket.r2.dev/screenshot-1712345678901.png"
}

PDF 接口的 message 为 "PDF saved"；path 为可下载的公网 URL。
